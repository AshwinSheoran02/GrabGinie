Grab Genie

AI Agent for Multi-Service Intent Orchestration
GrabHack 2.0
⚠️ The Problem
Super-apps contain immense utility but require users to navigate across disconnected vertical silos (Food, Ride, Mart) to complete composite tasks. Discoverability relies on manual transaction flows.

Example Natural Request:

"Chocolate ice cream for me and a priority premium ride to the airport."

One Request → One Intelligent Plan

Voice / Text
→
Grab Genie
→
Orchestrated Cart
✨ The Solution
An intelligent orchestrator that transforms natural language into a structured, multi-service plan. It bridges the gap between user intent and Grab's expansive service ecosystem.
🚀 Key Capabilities
  • Cross-Service Orchestration: Combines Ride, Food, and Mart items simultaneously.
  • Contextual AI Reasoning: Powered by Gemini to infer default constraints (e.g. matching "airport" to a Ride intent).
  • Proactive Alternatives: Generates smart fallback recommendations directly on the UI.
  • Seamless Confirmation: Review the entire multi-service plan in one holistic view.
